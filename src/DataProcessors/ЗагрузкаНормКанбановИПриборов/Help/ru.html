ВНИМАНИЕ!

Первая колонка - это наименование спецификации в справочнике Номенклатура.

Остальные колонки - это тех. операции из выбранной в настройках группы тех. операций. Поиск тех. операций в выбранной группе идёт по коду тех. операции, который должен совпадать с номером колонки минус один в файле.

Т.е. колонка номер 2 - это тех. операция с кодом 1, колонка номер 3 - это тех. операция с кодом 2 и т. д.
